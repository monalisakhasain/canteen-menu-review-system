# Canteen Menu and Review System

A dynamic web-based application that displays the daily cafeteria menu and allows students to rate dishes and leave reviews.
This system enables students to view available food items, check ratings, and provide feedback. It also includes an administrative module for managing dishes and moderating user reviews.

---

## Description

The Canteen Menu and Review System is designed to improve communication between students and the cafeteria. It provides a centralized platform where users can explore menu items, evaluate dishes through ratings, and share their experiences through reviews.
The system also includes an admin dashboard that allows efficient management of menu items and user-generated content.

---

## Prerequisites

Ensure the following software/tools are installed:

* WAMP Server (Apache, MySQL, PHP)
* Web Browser (Google Chrome or equivalent)
* Git (optional, for cloning the repository)

---

## Installation Steps

### 1. Clone or Download the Repository

```bash
git clone https://github.com/monalisakhasain/canteen-menu-review-system.git
```

Alternatively, download the ZIP file and extract it.

---

### 2. Move Project to WAMP Directory

Move the project folder to:

```
C:\wamp\www\
```

---

### 3. Start WAMP Server

Start the WAMP server and ensure:

* Apache is running
* MySQL is running

---

### 4. Create Database

1. Open phpMyAdmin
2. Create a new database named:

```
canteen_db
```

---

### 5. Import Database

1. Select the `canteen_db` database
2. Click on the "Import" tab
3. Choose the file:

```
database.sql
```

4. Click "Go" to import

---

### 6. Run the Application

Open a web browser and navigate to:

```
http://localhost/canteen_project
```
